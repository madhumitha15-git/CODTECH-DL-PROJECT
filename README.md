# CODTECH-DL-PROJECT
# Brain MRI Classification Using CNN

## Overview

This project uses Deep Learning and Convolutional Neural Networks (CNN) to classify brain MRI images into different categories.

The goal of this project is to build an image classification model that can analyze MRI scans and predict the corresponding class.

## Technologies Used

* Python
* TensorFlow / Keras
* CNN (Convolutional Neural Network)
* OpenCV
* NumPy
* Matplotlib
* Scikit-learn

## Project Workflow

1. Dataset collection and loading
2. Image preprocessing
3. Image resizing and normalization
4. Data augmentation
5. CNN model building
6. Model training and validation
7. Model evaluation
8. Prediction on new MRI images

## Model Architecture

The CNN model contains:

* Convolution layers for feature extraction
* Pooling layers for dimensionality reduction
* Dense layers for classification
* Softmax activation for multi-class prediction

## Features

* Automatic MRI image classification
* Deep learning based feature extraction
* Image preprocessing pipeline
* Model evaluation using performance metrics

## Results

The trained CNN model learns patterns from MRI images and predicts the class of unseen images.
